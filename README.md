# TIL

1. Set up:

   ```bash
   brew install pnpm

   pnpm install

   pnpm dev
   ```

2. Use `pnpm new <title>` to create new posts, or add your posts to `src/content/posts`.

## Commands

- `pnpm new <title>` - Create a new post (use `_title` for drafts)
- `pnpm update-theme` - Update the theme to the latest version
